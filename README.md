#### 对于熊洞内的进攻与防守（无人守）

* 进攻方 - 犀牛如何进熊洞

    ```txt
    桃花岛熊洞有两个洞口，两个洞口犀牛都可以进去。熊洞外的箭塔根据实际情况或用投石车或用强弩台或用犀牛拆掉。
    大门处的洞犀牛直接进去就行，悬崖边的洞有两条路可以进去。
    ```

  * 犀牛入熊洞路线一(正门)

  * 犀牛入熊洞路线三（悬崖边）

    ![犀牛入洞三](https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExZXVzcHB2cXo1cWVpbW9haWExaXYyMGsxZnk0bDR6OHFwNDk5dnVxZyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/G5hoPNxQOfi70Tr86H/giphy-downsized-large.gif)

* 防守方 - 如何防止犀牛进洞

  * 防犀牛入洞一（正门）

    ```txt
    正门洞口处可以摆一块不超出地面的大型地基，犀牛拆不到，进不来。只能人用武器或强弩先把这块地基拆了犀牛才能进。
    ```

    ![熊洞防御1](/res/pvp/bearcave_defense_1.png)

    ```txt
    然后可以再摆5个箭塔对着洞口，此时犀牛要想进洞，就得人吸完箭，拆完大型地基或者强驽台点掉箭塔才能进。此时有人守基本无敌，但是这里讨论的是无人守的情况，所以还不够。地面要再加柱子和矛戟拒马作为犀牛前进路上障碍。
    ```

    ![熊洞防御2](/res/pvp/bearcave_defense_2.png)

    ```txt
    如上图，洞口对于犀牛来说是封死了，但是如果人要走这个洞口可以在柱子上放天花板，不过这样犀牛有跳上去的风险，但也可以借此做个陷阱，比如两个天花板间留点空隙，让犀牛卡在空隙的矛戟拒马上。这里是不加路，直接走悬崖边洞。
    ```

    ```txt
    这还不够，虽然难拆，但还是能拆。所以继续加防御。洞内不用地基。用金属柱子和木制天花板构建地面，然后继续用金属柱子和矛戟拒马铺满地面，理想情况下整个洞犀牛都举步维艰，但又不影响正常活动。
    ```

    ![熊洞防御3](/res/pvp/bearcave_defense_3.png)

    ```txt
    贴墙箭塔放满，在第一层天花板上放强驽台穿透箭打驯养单位，第二层天花板下挂喷火器及诸葛连弩。
    ```

    ![熊洞防御4](/res/pvp/bearcave_defense_4.png)

    ```txt
    第二层天花板边缘放火油架，一个是打伤害，另一个是听响声，响声要与犀牛的声音抗衡。
    ```

    ![熊洞防御5](/res/pvp/bearcave_defense_5.png)

    ```txt
    第二层天花板上可以放弹药架和狼哨窝，狼哨兵可以从第二层天花板直接跳到洞口攻击敌人，而敌人因为柱子和拒马寸步难行。
    ```

    ![熊洞防御6](/res/pvp/bearcave_defense_6.png)

  * 防犀牛入洞二（悬崖边）

    ```txt
    悬崖边的洞跟大门处的洞区别在于，大门处洞犀牛可以退，悬崖边的洞犀牛进来了基本出不去，不能拆一会儿恢复一会儿。注意不能拍墙挡箭塔，犀牛必须有来无回。
    ```

    ![熊洞防御7](/res/pvp/bearcave_defense_7.png)

#### 熊洞的价值

* 防御

  ```txt

## 帝国商业

<details open>
  <summary>展开</summary>
</br>

```txt
这里的商业指集市交易，集市交易的商品比较值得注意的有三种，一种是有人定期扫货的，一种是有人经常扫货的，还有就是由于突发事件扫货的。比较有价值的是前两种，第三种需要经常在线才能做出应对。
```

* 定期扫货商品举例

  ```txt
  此类商品可能是一两天两三天被人扫货一次，时间没到的时候集市上可能卖20铜没人买，等到扫货的时候50铜的都被人买光，比如玉矿。
  ```

* 经常扫货商品举例

  ```txt
  此类商品表现为价格合适秒没，价格较高的会一直挂着没人买，比如草绳作为很多器械的原材料，需求极大而茅草获取费力。
  ```

* 突发事件扫货举例

  ```txt
  此类商品表现为价格高了基本不会有人买，但是可能突然被人买光了。比如箭塔，平时没存货，被拆了就必须购买。
  ```

### 师出有名

```txt
给自己叠一层正义buff，控制舆论对对手形成心理压制，同时加强己方凝聚力。比如找匹黑马起个名字叫1400不怕老虎追，然后找个合适的地方扔那边等人杀，然后就可以名正言顺打过去。
```

### 诱敌深入

```txt
一次意外中发现箭塔反应还是慢了（似乎是因为无人到有人需要一段时间加载），可以借此诱敌深入。在箭塔包围中放一个诱饵，比如马，敌方如果看到诱饵觉得没箭射他就一直冲过去就会发现跑得太深入了，实际上箭塔范围早就包括他了，只是加载慢了，这时候想跑就难了。
```

### 釜底抽薪

1. 第一步是拆掉整个区所有床，可以通过长时间（3周左右）爆服让界碑自行腐坏或者拆掉所有界碑让床都消失进来的玩家只能在复活点复活。

2. 把所有的复活点或用高墙围起来或用货车等卡死或用箭塔守，让玩家只能在复活点不断重生，做到了就可以轻而易举霸占整个区。

### 合纵连横

> 多个小势力如何用极低乃至零成本解决大势力的策略。

* 大致做法

  ```txt
  丢界碑，满地图丢界碑加箭塔，如果界碑多随便丢，少一点可以挑地方丢，想象一下，可能只要两个人，十几个界碑加箭塔大势力在这个区可能就玩不了了。
  ```

* 可行性分析

  1. 成本问题

      ```txt
      选用砖石箭塔，成本低廉的同时兼具威力。只需要铜锭、铁锭、铁矿和草绳等低级资源，且数量要求少，可以毫不费力的生产出大量界碑，砖石箭塔和哨箭。但是要拆就没那么容易了，攻城器械别说成本，制作时间就非常长，特别是弹药。两相对比，材料成本完全是以一换十换百，时间成本更是不成正比。器械其实也花不了多少钱，主要的目的是在消耗敌方的时间和精力。

  2. 被拆的时候怎么处理

      ```txt
      不要浪费时间精力守，这个就是给他们拆的，不拆都不行，以此消耗时间精力，界碑没了找合适的机会（比如至少得等对面把犀牛骑回去，要拆让他再骑出来）原地或换个地方加箭塔再起，砖石箭塔不管被拆没被拆都不用管，不拆就留在那继续射几个小时。
      ```
  
  3. 如果大势力也这么做怎么办

      ```txt
      大势力做不到这一点，大势力的优势是人多不管是势力技能还是势力资源都较好，这些东西此时就成了掣肘，谁愿意出去单立一个势力，甚至很多人就是不想单立一个势力才加的大势力。如果是开小号那么就更费劲了。假设主公分配任务，那势力就不仅仅是被赶出这个区，可能直接从内部开始直接分崩离析。
      ```

  4. 觉得还不够怎么加强攻势

      ```txt
      一周花几万铜给界碑上保护时间，养一个早睡早起的习惯，在早上放界碑，不设防时间设置为早上，其它时间段一分钟不留。这样子大势力要么每天都要起早满地图检查满地图起器械拆，拆完还啥都没有，不拆就进保了，然后开始全天折磨，根本玩不了。
     ```

* 为什么要采取这个策略

  ![论侵略者](/res/art_of_war/论侵略者.png)

  ```txt
  不打疼侵略者，你就永远是待宰的羔羊，养肥了就吃你肉。不要抱有幻想，谈什么都没用，因为目的都是为了宰你，不给就打，打完还要让你自己给，然后养肥了继续打。所以必须打疼他们，并且你不缺乏对抗的能力，你缺少的只是这篇兵法。
  ```
